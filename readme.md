## What
A full stack prototype (React + Next.js API routes) implementing the Hotel Room Reservation assignment.

## Run locally
1. Install:
npm install

2. Dev:
npm run dev
Open http://localhost:3000

3. Build & start:
npm run build
npm start

4. Run tests:
npm test


## Deploy to Vercel
1. Push the repository to GitHub.
2. Go to https://vercel.com/new and import the GitHub repo.
3. Use the default settings; Vercel will detect Next.js and set up builds.
4. Deploy. The app will be live at `<your-app>.vercel.app`.
